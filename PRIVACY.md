# Privacy Policy

Application Assist stores profile data that the user enters into the extension popup using Chrome storage.

## Data handled by the extension

- Name
- Email
- Phone number
- City
- Resume summary
- Resume link
- Resume file label
- Assistant pause state related to CAPTCHA or robot-check detection

## How data is used

- Data is used locally by the extension to help the user review and fill supported form fields.
- Data is not transmitted by the extension to any external server controlled by this project.
- The extension does not collect analytics or tracking data.

## File uploads

- The extension does not upload resume files.
- Resume file selection is used only to remember the file name for manual upload reminders.

## User control

- Users can change or overwrite their saved profile at any time through the popup.
- Users can remove the extension to stop all storage and usage.
